# IPS Colombia — Análisis por Nivel de Complejidad

## Descripción
Análisis de la distribución de Instituciones Prestadoras de Salud (IPS) 
en Colombia según su nivel de complejidad. Se explora la cobertura 
regional y las desigualdades en acceso a atención especializada en el país.

## Integrantes
- Laura Juliana Salazar — análisis de datos
- Maria Alejandra Rodriguez — desarrollo de la app

## Dataset
- Fuente: datos.gov.co
- Nombre: Listado de IPS en Colombia según su nivel de complejidad
- Link: https://www.datos.gov.co
- Variables: 27 variables · 11.466 registros

## Tecnologías usadas
- Python · PyCaret · Streamlit · GitHub

## Estructura del repositorio
- notebook.ipynb → análisis y modelos
- app.py → aplicación Streamlit
- requirements.txt → librerías necesarias
- datos/ → dataset del proyecto

## Cómo correr la app
1. pip install -r requirements.txt
2. streamlit run app.py
